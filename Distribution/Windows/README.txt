Приложение находится в папке Prototype. В той же папке лежат файлы init.xml и init2.xml.
init.xml - содержит настройки для имитационной модели
init2.xml - содержит настройки для подсистемы мониторинга

Инструкция по установке:
1. Установить пакет net-snmp (установочный файл находится в папке SNMP)
2. Добавить в переменную окружения Path путь к папке, в которую установились файлы snmpget.exe и snmpwalk.exe

Сам прототип не требует установки. Для запуска необходимо запустить исполняемый файт Prototype.exe из папке Prototype.
